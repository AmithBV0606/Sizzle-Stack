# Sizzle-Stack

A simple website for Burger shop named Sizzle Stack, built using HTML, Tailwindcss and JavaScript.

<h2><u>Technologies used</u> : </h2>

<ol>
  <li>HTML</li>
  <li>TailwindCSS</li>
  <li>JavaScript</li>
  <li>Swiper-js</li>
  <li>ScrollReveal-js</li>
</ol>
